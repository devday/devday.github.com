A) HTML Structure - top

This theme is a responsive one page layout with one column made on Bootstrap 3.0 foundation so please refer to original docs for more info (http://getbootstrap.com/css/). If you would like to place or make a full screen width element - please place it outside of this div. The general template structure is the same throughout the template.

The template consists of:

  1. Ventcamp_HTML/index.html - main template with photo background
  2. Ventcamp_HTML/index-color.html - main template with solid background
  3. Ventcamp_HTML/index-vid.html - main template with video background
  4. Ventcamp_HTML/coming-soon.html - coming soon page
  5. Ventcamp_HTML/register.html - registration page
  6. Ventcamp_HTML/icons-reference.html - icon class names
  7. Ventcamp_HTML/elements.html - SWAPP design elements
  8. Sources/Ventcamp_PSD.psd - main PSD template

All of the boxed blocks within the main content area are nested within a div with an class of "container" Here is the general structure.

If you would like to edit the color preset of the theme you should do "Replace all" this color #ff7701 with the desired one or use any of those predefined ones :

D) Registration page - top

The only thing you need to do to setup the registration form is to change this file:

  1. HTML/process-form.php - add your email address, edit fields
  2. HTML/register.html - edit form fields + edit JavaScript found on the same page below

The form is validated and then submitted on the register.html page which then sends the data to process-form.php file that just gets the variables and forards the data to your email

E) Coming soon page - top

The only thing you need to do to setup the mailchimp form is to change this file:

  1. Ventcamp_HTML/coming-soon.html - add your date at line #137 and it's ready!

But please refer to the original script author if you need a deeper insight - http://stackoverflow.com/questions/5025455/some-basic-mailchimp-api-examples-required

F) Mailchimp form - top

The only thing you need to do to setup the mailchimp form is to change this file:

  1. assets/mailchimp/inc/store-address.php - (add Mailchimp API key at line #24 and subscription list ID at line #28)

But please refer to the original script author if you need a deeper insight - http://stackoverflow.com/questions/5025455/some-basic-mailchimp-api-examples-required
